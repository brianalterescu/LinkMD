# LinkMD

LinkMD is a specialized automation and productivity suite designed for **Allen Medical Associates**. This tool bridges the gap between Updox and PracticeFusion EHR, streamlining administrative tasks and reducing manual documentation time for clinical staff.

## Core Features

* **EHR Automation Workflow:** Automates the synchronization of confirmed appointments and messages from Updox directly into the PracticeFusion web interface, eliminating manual status updates.
* **Referral NPI Lookup:** Integrated real-time lookup via the CMS NPI Registry. Instantly retrieve provider information (address, phone, fax, specialization) and copy a perfectly formatted referral message to your clipboard for your referral department.
* **Clinical Document Generation:** A built-in note generator that allows staff to quickly create and print standardized letters, including:
    * **Custom Restriction Notes:** For work or school limitations.
    * **General Visit Notes:** For proof of attendance.
    * **Return to Work/School Notes:** With automated date formatting and custom return-to-office timelines.

## Workflow Guide

1.  **Prep Updox:** Navigate to your Updox confirmation page and scroll to the bottom to ensure all data is loaded into browser memory.
2.  **Scrape & Sync:** Use the LinkMD extension to capture confirmation codes and trigger the automated matching engine within your PracticeFusion Agenda.
3.  **Referral Assistance:** Simply enter any 10-digit NPI number in the LinkMD popup to generate a professional referral contact message instantly.
4.  **Note Generation:** Select your letter type, enter patient details, and generate a print-ready document in seconds.

## Installation

1.  **Download** the latest release ZIP file.
2.  **Extract** the folder to a permanent location on your machine.
3.  Open Chrome and navigate to `chrome://extensions`.
4.  Enable **Developer Mode** in the top-right corner.
5.  Click **Load Unpacked** and select the LinkMD folder.
6.  **Pin** the extension to your toolbar for quick access.
